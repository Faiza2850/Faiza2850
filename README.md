## 🚀 Featured Projects

### 🧠 Deep Learning & NLP Research

> ### **C2DB-T — Multi-Label Text Classification** | *(Submitted for Journal Review, 2026)*
> * **The Architecture:** Developed an NLP framework integrating a RoBERTa encoder, Distribution-Balanced loss reweighting, a GNN-based Label Correlation Module, and an Adaptive Threshold layer.
> * **The Impact:** Addressed severe class imbalance and label correlation constraints, achieving a **+5.72% Macro-F1** improvement on EurLex and **+4.3%** on a domain-specific biotech corpus.

* **Mini-AI Scientist — Autonomous Photonic Circuit Discovery (2026)**
  * Built a JAX-based differentiable simulator of photonic interferometer circuits using gradient-based optimization (`Adam`/`optax`) to autonomously discover target quantum interference patterns.
  * Integrated an interpretability layer utilizing parameter-sensitivity analysis alongside an interactive UI for real-time validation.
* **Multi-Label Cyberbullying & Online Harassment Detection Engine (2025)**
  * Fine-tuned `DistilBERT-base-uncased` in PyTorch to classify multi-label toxic comments across a dataset of 159,000+ entries.
  * Overcame extreme class imbalances to achieve a robust **97.2% validation accuracy** and a 0.76 micro F1-score.
